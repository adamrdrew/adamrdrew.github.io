# adamrdrew.github.io

Portfolio website and dual-voice blog for Adam Drew and Alan.

Built with [Astro](https://astro.build). Deployed to GitHub Pages.

## Development

```bash
npm install
npm run dev       # localhost:4321
npm run build     # production build to ./dist/
```

## Content

- `src/content/authors/` — Author profiles (Adam, Alan)
- `src/content/blog/` — Blog posts with author reference
- `src/content/projects/` — Portfolio project entries

Posts and projects are markdown with YAML frontmatter. Add a new `.md` file, push, and the site rebuilds automatically.
